## 本地部署
<Callout type="info" emoji="ℹ️">
  **Prerequisite** You should have installed Node.js (version 18.10.0 or
  higher).
</Callout>

<Steps>
  ### 步骤一：克隆项目

  ```bash
  git clone https://github.com/AprilNEA/ChatGPT-Admin-Web.git
  git checkout v3
  cd ChatGPT-Admin-Web
  ```

  ### 步骤二：安装依赖

  ``` bash pnpm
  pnpm install
  ```

  ### 步骤三：配置项目

  在目录的根目录参照一下`config.json`填写相关配置
  ```json
  {
    "mode": "nginx",
    "title": "ChatGPT Admin Web",
    "postgres": {
    "url": "postgres://postgres:t@localhost:5433/postgres"
    },
    "redis": {
      "enable": false
    },
    "port": {
      "frontend": 3000,
      "backend": 3001
    },
    "jwt": {
      "algorithm": "HS256",
      "secret": "jRn7UY9uub24"
    },
    "openai": {
      "baseUrl": "https://api.openai.com/v1",
      "keys": [
        "sk-ffffff"
      ]
    },
    "email": {
      "use": "resend",
      "domain": "exmaple.com",
      "resend": {
        "apiKey": "ffffff"
      }
    },
    "sms": {
      "use": "uni",
      "uni": {
        "signature": "XX科技",
        "apiKey": "ffffff",
        "apiSecret": ""
      }
    },
    "payment": {
      "use": "xunhu",
      "xunhu": {
        "wapName": "XX科技",
        "appId": "123456",
        "appSecret": "ffffff",
        "notifyUrl": "ffffff",
        "returnUrl": "ffffff"
      }
    }
  }

  ```

  ### 步骤四：构建项目
  ```bash
  turbo run build
  ```

  ### 步骤五：启动项目
  ```bash
  turbo run start
  ```
  项目默认运行在 `http://localhost:3000`。

</Steps>
